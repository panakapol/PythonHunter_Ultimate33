1. BASICS (พื้นฐานและตัวแปร)

Python เป็นภาษาที่เน้นความอ่านง่าย การประกาศตัวแปรทำได้ทันทีโดยไม่ต้องระบุชนิดข้อมูล แต่เป็นภาษาที่ Case-sensitive (พิมพ์เล็ก-ใหญ่ มีผลต่างกัน)

  • print(...) : แสดงผลข้อมูลออกหน้าจอ Console
  • input(...) : รอรับค่าจากแป้นพิมพ์ (ข้อมูลจะเป็น String เสมอ)
  • int(...) : แปลงข้อมูลให้เป็นเลขจำนวนเต็ม (Integer)
  • float(...) : แปลงข้อมูลให้เป็นเลขทศนิยม
  • str(...) : แปลงข้อมูลให้เป็นข้อความ (String)
  • type(...) : ใช้ตรวจสอบชนิดของข้อมูล
  • del : ลบตัวแปรออกจากหน่วยความจำ
  • True / False : ค่าความจริง (Boolean) ต้องขึ้นต้นด้วยตัวพิมพ์ใหญ่

2. LOGIC (ตรรกะและเงื่อนไข)

หัวใจสำคัญคือ การย่อหน้า (Indentation) เพื่อบอกขอบเขตของบล็อกคำสั่ง แทนการใช้วงเล็บปีกกา

  • if : ถ้าเงื่อนไขเป็น "จริง" ให้ทำคำสั่งในบล็อกที่ย่อหน้า
  • elif : (Else If) ตรวจสอบเงื่อนไขถัดมา หาก if ด้านบนเป็นเท็จ
  • else : กรณีสุดท้าย จะทำงานเมื่อทุกเงื่อนไขเป็นเท็จ
  • == : เครื่องหมาย "เท่ากับ" (เปรียบเทียบค่า)
  • != : เครื่องหมาย "ไม่เท่ากับ" (Not Equal)
  • and / or : ตรรกะ "และ" (ต้องจริงคู่) / ตรรกะ "หรือ" (จริงแค่อันเดียว)
  • is / is not : ตรวจสอบ Object หรือเช็คค่า None อย่างปลอดภัย

3. LOOPS (การวนซ้ำ)

  • for i in ... : ใช้เมื่อรู้จำนวนรอบที่แน่นอน หรือวนซ้ำสมาชิก List
  • range(start, stop) : สร้างลำดับตัวเลข (stop จะไม่ถูกรวม)
  • while : ทำงานวนซ้ำตราบใดที่เงื่อนไขยังเป็น True
  • break : สั่ง "หยุด" และออกจาก Loop ทันที
  • continue : สั่ง "ข้าม" การทำงานในรอบนั้นเพื่อไปเริ่มรอบถัดไปทันที
  • enumerate() : ฟังก์ชันช่วยวนลูปที่คืนค่าทั้ง Index และ Value

4. DATA STRUCTURES (โครงสร้างข้อมูล)

  • List [...] : เก็บแบบลำดับ แก้ไขได้
  • .append(x) / .pop() : เพิ่มข้อมูลต่อท้าย / ดึงตัวสุดท้ายออก
  • Dict {key: value} : เก็บข้อมูลแบบคู่ ไม่เรียงลำดับ เข้าถึงด้วย Key
  • .keys() / .items() : ดึง Key ทั้งหมด / ดึงทั้งคู่ Key-Value
  • Set (...) : ไม่เก็บค่าซ้ำ ใช้เมธอด .add() เพื่อเพิ่มข้อมูล

5. FUNC (ฟังก์ชันและโมดูล)

  • def name(): : ประกาศสร้างฟังก์ชัน
  • return : ส่งค่าผลลัพธ์กลับและจบการทำงานทันที
  • import ... as ... : นำเข้า Module นอกมาใช้ และตั้งชื่อเล่น (Alias)
  • *args / **kwargs : รับพารามิเตอร์จำนวนมากเข้ามาเป็น Tuple หรือ Dict

6. OOP (การเขียนโปรแกรมเชิงวัตถุ)

  • class Name: : การประกาศสร้างแม่พิมพ์ (Blueprint)
  • __init__ : Constructor ที่จะทำงานอัตโนมัติเมื่อสร้าง Object
  • self : ตัวแปรใช้แทน "ตัวเอง" ภายในคลาส
  • super() : เรียกใช้เมธอดจากคลาสแม่ (Inheritance)

7. DATABASE & FILE I/O

  • sqlite3.connect() : เชื่อมต่อ Database และ .execute() สั่งรัน SQL
  • .commit() / .close() : บันทึกการเปลี่ยนแปลง / ปิดการเชื่อมต่อ Database
  • open('f.txt', 'r') : เปิดอ่านไฟล์ .read() อ่านทั้งหมด .write() เขียนข้อมูล

9. SURVIVAL (Advanced - โหดพิเศษ)

  • Comprehension : การสร้าง List ในบรรทัดเดียว [x for x in r]
  • lambda : ฟังก์ชันขนาดสั้นแบบไม่มีชื่อ
  • yield : สร้าง Generator เพื่อประหยัด Memory ลูป
  • nonlocal : เข้าถึงหรือแก้ไขตัวแปรที่อยู่ในขอบเขตชั้นนอก
  • := (Walrus) : ทำ Assignment ภายใน Expression ไปในตัว
